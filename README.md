## 💡 What I care about

- **Architecture first** — I'd rather spend time getting the structure right than refactoring later
- **Compile-time safety** — if the compiler can catch it, it shouldn't reach runtime
- **Automation** — if something can be automated, it will be
- **Modularity** — code should be easy to change, test, and replace
